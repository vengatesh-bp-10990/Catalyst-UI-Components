Installation

Get started with Catalyst UI Components in your sLyte project.

Install via npm

Terminal 
npm install catalyst-ui-components --registry http://cm-npmregistry

Import tokens in your HTML

HTML 
<link rel="stylesheet" href="node_modules/catalyst-ui-components/theme/zcat-tokens.css">
<link rel="stylesheet" href="node_modules/catalyst-ui-components/theme/zcat-typography.css">
<link rel="stylesheet" href="node_modules/catalyst-ui-components/styles/app.css">

Usage

Use ZCAT components in your sLyte templates or plain HTML with CSS classes.

sLyte Component

HTML 
<zcat-button-comp lt-prop-variant="fill" lt-prop-color="primary" lt-prop-size="md">
</zcat-button-comp>

CSS Class Usage

HTML 
<button class="zcat-btn zcat-btn--fill zcat-btn--primary zcat-btn--md">
  Submit
</button>
